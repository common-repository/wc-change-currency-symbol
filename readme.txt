This plugin allows you to change the currency symbol from the default Woocommerce Symbol to a custom symbol.

== Description ==
This plugin allows you to change the currency symbol from the default Woocommerce Symbol to a custom symbol. After activating the plugin, you can change the symbol through the Woocommerce settings page. 

== Installation ==
1. Upload plugin zip file or Install directly from WordPress plugin directory
2. Activate the plugin
